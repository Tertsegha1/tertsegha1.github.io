New accounts can only be created with a valid passcode below. Each passcode works only once — it deactivates itself the moment someone registers with it, so the same code can't be reused or passed around. Generate one per student (e.g. one for each name on your register) and hand them out individually, or generate a batch in one go below. Revoke an unused code any time to stop it working.

💡 This track assumes Python basics. Only issue passcodes here to students who've already completed at least the Python Beginner level — there's no automatic check for this, it's on you to gate it at the point of handing out a code.
